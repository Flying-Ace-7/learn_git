## Tagging

We can use tags to tag a specfic commit. To tag the commit you previously made: Use

```git tag <tag_name>```


for a lightweight tag, with jsut the tag name.

If a more detailed tag is needed, we can use:
```git tag -a <tag_name> -m "message"```



To list all the stored tags in a repo, use 
``` git tag```
